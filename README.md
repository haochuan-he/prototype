# 🍅 Pomodoro Timer

A clean, modern, frontend-only Pomodoro timer app.

**Features:**
- 🍅 Pomodoro / ☕ Short Break / 🌙 Long Break modes
- Animated SVG circular progress ring
- Web Audio API chime on session complete (no external files)
- Desktop notifications support
- Customizable durations & rounds (saved to localStorage)
- Keyboard shortcuts: `Space` to start/pause, `Esc` to close settings

## 🚀 Deploy to GitHub Pages

1. Push this folder's contents to your repository's `main` branch (at the repo root).
2. Go to **Settings → Pages → Source → Deploy from branch → main / (root)**.
3. Visit `https://<your-username>.github.io/<repo-name>/` — done!

> The app is purely static (HTML + CSS + JS), no build step needed.
